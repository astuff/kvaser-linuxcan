CAN bus API (CANlib)

CANlib is a software product from KVASER that provides support for a wide range of Kvasers CAN products. The CANlib API provides the application programmer with a quick and easy access to the CAN network while supporting a queue-oriented programming model.

CANlib supports both 32-bit and 64-bit programs under Windows and Linux. For more detailed information about supported platforms, please contact Kvaser Support.
